# Chicago HealthCheck — Landing Page

Responsive landing page built from a Figma design as a test task for the **Web Developer** position at Alpha Efficiency.

🔗 **Live preview:** https://alpha-efficiency-test-six.vercel.app

## Built with

- **HTML5** — semantic, accessible markup
- **CSS3** — mobile-first, custom properties, CSS Grid & Flexbox
- **Vanilla JavaScript** — no frameworks or libraries

## Features

- **Fully responsive** from 1920px down to 320px (breakpoints at 992px and 600px)
- **Burger menu** — accessible mobile navigation drawer with animated icon
- **Live countdown timer** — counts down to the offer deadline, persisted in `localStorage`
- **FAQ accordion** — single-open, smooth expand/collapse
- Accessible: ARIA attributes, keyboard-friendly, `prefers-reduced-motion` support
